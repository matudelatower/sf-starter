Symfony Starter Template
=========

A Symfony project created on June 12, 2020, 17:33.

- Symfony 5.1 + webpack Encore
- AdminLTE 3.0.5


Install:

1. Install php dependencies: 

    `$ composer install`

2. Install js dependencies

    `$ npm install`

3. Copy `.env` to `.env.local` and set your enviroment    

4. `npm run build`

5. Enjoy
